This HTML Cheat Sheet is For You!
  1. HTML Boilerplate : This is basic HTML boilerplate code 
    <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
    
</body>
</html>
  2. Style Tag : This is basic style tag 
    <style>
h1 {color:red;}
p {color:blue;}
</style>
  3. Script Tag : This is basic script tag 
    <script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>
  4. Date : Find date in JavaScript 
    const d = new Date();
d.getTime();
  5. Time : Find time in JavaScript 
    const d = new Date();
let time = d.getTime();
  6. SetInterval : Update your code time to time 
    setInterval(function () {element.innerHTML += "Hello"}, 1000);
  7. SetTimeout : Update your 
    const myTimeout = setTimeout(myGreeting, 5000);
